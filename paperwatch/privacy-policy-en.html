Privacy Policy

Last Updated: January 12, 2026

Introduction

PaperWatch ("the App") respects your privacy and is committed to protecting your personal information. This Privacy Policy explains what information we collect and how we use it.

Information We Collect

1. Anonymous User ID

The App uses Firebase Authentication to perform anonymous authentication. A randomly generated anonymous user ID is created during this process. This ID is not linked to any personally identifiable information and is used solely to provide app functionality.

2. Device Token

We collect Firebase Cloud Messaging (FCM) device tokens to send push notifications. This token identifies your device and is not linked to personal information.

3. User Preferences

The following settings are stored in Firestore:

  • Subscribed arXiv categories
  • Configured keywords
  • Notification time settings
  • Language preferences

4. Paper Status

We store the read and favorite status of papers marked by users.

How We Use Information

We use collected information only for the following purposes:

  1. Providing App Features: Filtering and displaying papers based on user preferences
  2. Sending Push Notifications: Alerting users about new papers
  3. Improving User Experience: Syncing read and favorite status

Third-Party Services

The App uses the following third-party services:

Firebase (Google LLC)

Used for authentication, database, and push notifications
Google Privacy Policy

Anthropic Claude API

Used to generate Japanese summaries of papers. Only paper metadata (title, authors, abstract) is sent. No personal information is transmitted.
Anthropic Privacy Policy

arXiv

Used to retrieve paper metadata. Operated in accordance with arXiv API Terms of Use.
arXiv Privacy Policy

About AI-Generated Content

The Japanese summaries displayed in this App are automatically generated by Anthropic's Claude AI. These summaries are for reference only and may not accurately reflect the content of the original papers. Please refer to the original papers for accurate information.

Data Storage and Deletion

  • User data is stored on Firebase servers
  • Uninstalling the app does not automatically delete server-side data
  • If you wish to delete your data, please contact the developer

Tracking and Advertising

This App:

  • Does not track user behavior
  • Does not display advertisements
  • Does not sell collected data to third parties

Children's Privacy

This App is not intended to collect information from children under 13 years of age. If we learn that a child under 13 has provided personal information, we will promptly delete it.

Changes to This Privacy Policy

This Privacy Policy may be changed without notice. If changes are made, we will update this page and revise the last updated date.
